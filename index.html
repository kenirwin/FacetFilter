FacetFilter

FacetFilter is a client-side tool for creating a facet-based sorting/limiting interface for exploring web-based content based on an array of JSON objects. The content format is customizable. It is ideal guiding users to content for small sets of data (e.g. dozens or hundreds of entries) that can be represented in a static JSON file.

Download

FacetFilter is available as an open-source solution on GitHub.

Demos

Because the content is natively in JSON format, it can be easily formatted in a variety of ways. These demos illustrate some different ways that the content can be presented:

Demo: Faces

An image-based demo that uses images and text to represent the content.

Demo: Colors & Numbers

A non-image based arrangement that demonstrates three different kinds of facets: string, number, and tag formats.

Demo: Unformatted

This demo illustrates the standard, unformatted appearance when an item format is not defined. This example uses only a little bit of CSS to clean up the output.
